# markitdown-quickstyle
这是微软的markitdown的便捷使用版本!

支持以下这些转换为markdown文档

| 分类                | 格式 / 扩展名     | 说明                                  |
| ------------------- | ----------------- | ------------------------------------- |
| 📄 **Office文档**    | `.docx`           | Word（主要支持）                      |
|                     | `.doc`            | Word旧格式（部分支持）                |
|                     | `.pptx`           | PowerPoint                            |
|                     | `.ppt`            | PPT旧格式（部分支持）                 |
|                     | `.xlsx`           | Excel                                 |
|                     | `.xls`            | Excel旧格式                           |
| 📑 **PDF**           | `.pdf`            | 文本提取                              |
| 📚 **电子书**        | `.epub`           | 电子书                                |
| 🖼️ **图片**          | `.jpg` / `.jpeg`  | 图片（EXIF + OCR）                    |
|                     | `.png`            | 图片                                  |
|                     | `.gif`            | 图片/动图（仅解析，不保留动画）       |
|                     | `.webp`           | 图片                                  |
|                     | `.bmp` / `.tiff`  | 高质量图像（通过扩展支持）            |
| 🎧 **音频**          | `.wav`            | 语音转文字                            |
|                     | `.mp3`            | 语音转文字                            |
|                     | `.m4a`            | 部分支持                              |
| 🌐 **网页/在线内容** | `.html` / `.htm`  | 网页                                  |
|                     | URL（HTTP/HTTPS） | 任意网页                              |
|                     | YouTube URL       | 提取字幕                              |
|                     | Wikipedia URL     | 页面解析                              |
|                     | RSS Feed          | 订阅内容                              |
| 📊 **数据文件**      | `.csv`            | 表格数据                              |
|                     | `.json`           | JSON                                  |
|                     | `.xml`            | XML                                   |
|                     | `.ipynb`          | Jupyter Notebook                      |
| 📦 **压缩包**        | `.zip`            | 自动遍历内部文件                      |
| 📧 **邮件**          | `.msg`            | Outlook邮件                           |
| 🧠 **高级扩展**      | 任意格式          | Azure Document Intelligence（可扩展） |
